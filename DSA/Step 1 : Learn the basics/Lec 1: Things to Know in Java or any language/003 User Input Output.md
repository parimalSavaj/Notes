## for input

```
import java.util.Scanner;

Scanner sc = new Scanner(System.in);

sc.nextInt();  // int
sc.nextLong(); // long
sc.nextFloat(); // float
sc.nextDouble(); // double
sc.nextBoolean(); // boolean
sc.next(); // String
sc.nextLine(); // multi line String

sc.close();
```

## for output

```
import java.util.Scanner;

Scanner sc = new Scanner(System.in);

System.out.println("Hello World"); // print and next line

System.out.print("Hello World"); // print without next line
```
